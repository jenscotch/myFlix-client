# myFlix-client
 
# Project Objective:
Using React, build the client-side for an app called myFlix based on its
existing server-side code (REST API and database).

# Libraries Utilized:
JavaScript, React, SASS, HTML

# UI Library:
Bootstrap

# Build Tool:
Parcel
